2.1. odtone::sap::sap Class Reference

Detailed Description

SAP Base Class

This class derive from boost::noncopyable in order to prohibit copy construction and copy assignment. Thus, odtone::sap::sap class has protected constructor and destructor members and should be used only as a base class.

This will be the base class to odtone::sap::user and odtone::sap::link as is documented later in this document.
