2.12. odtone::mih::tlv_type_ Class Reference

Detailed Description

TLV Type.

The MIH types support serialization and deserialization for the odtone::mih::iarchive and odtone::mih::oarchive, however those particular archives does not include the TLVs fields.

So odtone::mih::tlv_type_ is responsible to manage TLVs and offers mechanisms to serialize and deserialize them.

There are 2 types of this class. However, since they are similar, we will explain them generically:

  • odtone::mih::tlv_type_< T, TLV >
  • odtone::mih::tlv_type_< boost::optional< T >, TLV >
