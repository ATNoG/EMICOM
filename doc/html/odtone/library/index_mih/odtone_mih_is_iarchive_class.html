2.11. odtone::mih::is_iarchive Class Reference

Detailed Description

This class permits to check if a given class is an input archive (odtone::mih::iarchive).

It has only a boolean value that is true when the given class is an input archive (odtone::mih::iarchive), or false otherwise.
