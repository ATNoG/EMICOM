2.2. odtone::mih::message_helper Class Reference

Detailed Description

MIH Message helper.

Parsing/Generating MIH messages is done with helper classes (odtone::mih::request, odtone::mih::response, odtone::mih::indication) and operator overloading.

This will be the base class to odtone::mih::request, odtone::mih::response and odtone::mih::indication, as is documented later in this document.

This class derive from boost::noncopyable in order to prohibit copy construction and copy assignment. Thus, odtone::mih::message_helper class has protected constructor and destructor members and should be used only as a base class.
