2.11. odtone::mih::is_oarchive Class Reference

Detailed Description

This class permits to check if a given class is an output archive (odtone::mih::oarchive).

It has only a boolean value that is true when the given class is an output archive (odtone::mih::oarchive), or false otherwise.
