Getting Started

Extending

It is possible to modify the behaviour of the dummyserver when it receives a new message. To do this, the "static void process_message(meta_message_ptr& msg)" function must be modified.

E.g.:

Let's change the dummyserver to send the incomming message to specific destination (assuming that the mechanisms for sending messages have already been defined): 

static void process_message(meta_message_ptr& msg)
{
	std::cout << "MIH message: Service=" << msg->sid()
		  << " Opcode=" << msg->opcode()
		  << " Action=" << msg->aid()
		  << std::endl;

	sap.send(msg);
}

Running

Just run dummyserver and wait for incoming messages!
