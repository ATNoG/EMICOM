MIH User

The MIH_User specifies a media independent interface between the MIHF and upper layers of the mobility management protocol stack. The upper layers need to subscribe with the MIHF as users to receive MIHF-Generated events and also for link-layer events that originate at layers below the MIHF but are passed on to MIHF users through the MIHF. MIHF users directly send commands to the local MIHF using the service primitives of the MIH User.

This implementation of MIH User is very simple. It only support capability discover, event subscribe and link events primitives.
